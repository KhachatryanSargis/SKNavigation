## Package Overview

SKNavigation is a lightweight, type-safe SwiftUI navigation library built on the
Coordinator pattern. It depends on SKCore. iOS 17+ / macOS 14+, Swift 6.1,
strict concurrency.

## Key Areas

| Area | Purpose |
|---|---|
| **Coordinator** | `@Observable @MainActor` coordinators that drive all navigation |
| **Router** | Navigation state containers used by coordinators |
| **Route** | Protocol for defining destinations as enums |
| **Views** | `CoordinatedView` and `TabCoordinatedView` wiring |
| **CrossModule** | Navigate between feature modules without direct imports |
| **DeepLink** | Deep link handling protocol |
